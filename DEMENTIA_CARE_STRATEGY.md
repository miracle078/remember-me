## üé® CORE FEATURES (Priority Order)

### ‚úÖ MUST HAVE (Hours 0-8)

#### 1. **Photo Memory Bank with Face Recognition**
- Upload photos of family members
- Local face recognition (using Liquid vision model)
- Tag: "This is Sarah, your daughter"
- When patient opens app, shows photos with reminders
- **Privacy**: Photos never leave device, face models trained locally

**Use Case**: Patient sees someone, takes photo ‚Üí app says "This is your son Michael"

#### 2. **Voice Memory Journal**
- Record daily moments, conversations
- Local speech-to-text (Whisper or Qwen3)
- Searchable: "What did I talk about yesterday?"
- **Privacy**: Voice recordings stored encrypted locally

**Use Case**: Patient records: "Had coffee with grandson today, he told me about new job"

#### 3. **Smart Reminders with Context**
- Medication reminders
- Appointment reminders
- "You usually take walks at 3pm" (learned from patterns)
- Local notifications, no cloud sync

#### 4. **"Where Am I?" Safe Mode**
- GPS location tracking (local only)
- "You're at 123 Main St, your home address"
- Emergency contact info visible
- No location data sent to cloud unless emergency button pressed

### üåü NICE TO HAVE (Hours 9-14)

#### 5. **Conversational AI Companion**
- Chat interface using Qwen3
- Remembers past conversations (vector embeddings)
- Patient can ask: "Who is Michael?" ‚Üí "Your son, he works as a teacher, you talked about him yesterday"
- Provides companionship without judgment

#### 6. **Daily Routines Tracker**
- Learn patterns: "You usually eat breakfast at 8am"
- Gentle prompts if routine disrupted
- Pattern recognition helps identify behavior changes (early warning for caregivers)

#### 7. **Memory Search**
- Semantic search through all photos, voice notes, conversations
- "Show me memories with Sarah"
- "When did I last visit the doctor?"
- Timeline view of life events

### üíé POLISH (Hours 15-19)

#### 8. **Caregiver Dashboard (Optional Sharing)**
- Patient controls what caregiver sees
- Health metrics, medication adherence
- Behavior pattern alerts (if patient opts in)
- **Key**: Patient consent required, transparent about what's shared

#### 9. **Emergency Mode**
- If patient is lost/confused, can press SOS
- Sends location to designated caregiver (ONLY when button pressed)
- Shows patient their home address, caregiver contact

## üí° TECHNICAL IMPLEMENTATION

### Stack

**Framework**: Flutter
**Models**:
- Qwen3 (conversation, embeddings, text generation)
- Liquid (face recognition, photo analysis)
- Whisper (optional, for voice transcription)

**Database**: SQLite with:
- Photos table (with face embeddings)
- Voice notes table (with text transcriptions)
- Memories table (for manual entries)
- People table (family members with relationships)
- Reminders table

**Privacy Features**:
- AES-256 encryption for all local data
- Face recognition models trained on-device
- No network calls except optional emergency mode
- Transparent data sharing controls

### Memory Architecture (Track 1 Focus)

```
USER INTERACTION
    ‚Üì
[Take photo / Record voice / Ask question]
    ‚Üì
LOCAL AI PROCESSING
    ‚Üì
[Liquid: Face recognition] OR [Qwen3: Voice ‚Üí Text] OR [Qwen3: Q&A]
    ‚Üì
GENERATE EMBEDDINGS (Qwen3)
    ‚Üì
STORE IN LOCAL SQLITE (encrypted)
    ‚Üì
[Photos + Face embeddings] | [Voice notes + text] | [Conversations + context]

USER QUERIES
    ‚Üì
[Convert query to embedding]
    ‚Üì
SEMANTIC SEARCH (cosine similarity)
    ‚Üì
RETRIEVE RELEVANT MEMORIES
    ‚Üì
CONTEXT INJECTION to Qwen3
    ‚Üì
PERSONALIZED RESPONSE

"Who is Sarah?" ‚Üí Search embeddings ‚Üí Find: "Sarah, daughter, lives Seattle"
‚Üí "This is Sarah, your daughter. She lives in Seattle. You talked about
her yesterday when you mentioned her new job."
```

### Key Technical Differentiators

1. **Multi-modal Memory**: Photos + Voice + Text all searchable
2. **Relationship Mapping**: Knows "Sarah is daughter, Michael is son"
3. **Temporal Context**: "You saw her yesterday"
4. **Pattern Recognition**: "You usually eat lunch at noon"
5. **Privacy by Design**: Everything encrypted, nothing sent to cloud
